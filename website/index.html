❓ Common Questions

💰 Which AI API should I choose?

Claude: Best overall, great for everything. GPT-4: Excellent analysis. Gemini: Free tier available, great for video.

🔒 Is my data private?

Yes — 100%. Screenshots go ONLY to your chosen AI API. We never see your data. Your API key stays on your computer.

🔄 Can I use multiple AIs at the same time?

Yes. Connect Claude, OpenAI and Gemini together, run them in parallel, and switch between them anytime. Use different AIs for different tasks.

🎥 How do I analyze YouTube videos?

Video analysis requires a Gemini API key — only Gemini can read the full content of a video. Add a Gemini key in Settings, then paste any YouTube link.

🌐 Do I need internet?

Yes, internet is needed for AI features. Screenshots work offline, but AI analysis requires a connection.

💳 Will it cost money?

Only if you use paid AI APIs. All providers have free tiers. Gemini has a permanent free tier. Claude/OpenAI charge per usage.

Accessibility Statement

We are committed to making this site usable by everyone.

AI Screen Control is committed to ensuring digital accessibility for people with disabilities. We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1, Level AA, and we continually improve the experience for all users.

Accessibility features on this site

  • Full keyboard navigation, with a visible focus indicator on every control.
  • A "Skip to main content" link for keyboard and screen-reader users.
  • Semantic HTML landmarks and headings for assistive technologies.
  • Descriptive labels on buttons and interactive elements (ARIA).
  • Dialogs that can be closed with the Esc key and manage focus.
  • Respect for the "reduce motion" system setting.
  • Readable text sizes and high-contrast color combinations.
