Accessibility Statement

We are committed to making this site usable by everyone.

AI Screen Control is committed to ensuring digital accessibility for people with disabilities. We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1, Level AA, and we continually improve the experience for all users.

Accessibility features on this site

  • Full keyboard navigation, with a visible focus indicator on every control.
  • A "Skip to main content" link for keyboard and screen-reader users.
  • Semantic HTML landmarks and headings for assistive technologies.
  • Descriptive labels on buttons and interactive elements (ARIA).
  • Dialogs that can be closed with the Esc key and manage focus.
  • Respect for the "reduce motion" system setting.
  • Readable text sizes and high-contrast color combinations.

Need help or found a barrier?

If you encounter any accessibility barrier, or need assistance using this site or the app, please contact us and we will respond promptly:

📧 accessibility@aiscreencontrol.app
🐛 Report an issue on GitHub

This statement was last reviewed on July 17, 2026. Accessibility is an ongoing effort, and we welcome your feedback.
