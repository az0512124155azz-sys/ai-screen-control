❓ Common Questions

💰 Which AI API should I choose?

Claude: Best overall, great for everything. GPT-4: Excellent analysis. Gemini: Free tier available, great for video.

🔒 Is my data private?

Yes — 100%. Screenshots go ONLY to your chosen AI API. We never see your data. Your API key stays on your computer.

🔄 Can I use multiple AIs at the same time?

Yes. Connect Claude, OpenAI and Gemini together, run them in parallel, and switch between them anytime. Use different AIs for different tasks.

🎥 How do I analyze YouTube videos?

Video analysis requires a Gemini API key — only Gemini can read the full content of a video. Add a Gemini key in Settings, then paste any YouTube link.

🌐 Do I need internet?

Yes, internet is needed for AI features. Screenshots work offline, but AI analysis requires a connection.

💳 Will it cost money?

Only if you use paid AI APIs. All providers have free tiers. Gemini has a permanent free tier. Claude/OpenAI charge per usage.
