## [1.0.0] - 2024-01-15

### Added
- ✅ Initial release
- ✅ Cross-platform desktop app (Windows, macOS, Linux)
- ✅ Floating bubble UI with menu
- ✅ Screenshot capture
- ✅ AI chat integration (Claude API)
- ✅ Settings panel with API key configuration
- ✅ Multi-API support (Claude, OpenAI, Gemini)
- ✅ Installation website with setup guides
- ✅ Comprehensive documentation

### Features
- 📸 Screenshot capture with AI analysis
- 💬 Chat interface with message history
- ⚙️ Settings for API configuration
- 🎯 Floating bubble for quick access
- 🔒 Secure API key storage (localStorage)
- 📱 Responsive UI design
- 🌐 Multi-language AI support

### Technical
- Built with Tauri + React + Rust
- TypeScript for type safety
- Vite for fast development
- Cross-platform compatibility
- Open source (MIT License)

### Platform Support
- ✅ Windows 7+
- ✅ macOS 10.13+
- ✅ Linux (Ubuntu, Fedora, Arch)

### Known Limitations
- Screen control not yet available
- Video analysis requires Gemini API
- Local-only screenshot processing
- No offline mode
